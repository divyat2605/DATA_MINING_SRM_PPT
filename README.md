## 📚 Unit Summaries

### Unit 1 — Introduction to Data and Data Preprocessing
**File:** `Unit1.pptx`

Covers the foundational concepts of data and how it is prepared for analysis.

- What is data and why it matters
- Types of data: Numerical, Categorical, Structured, Unstructured
- Data objects and attribute types: Nominal, Ordinal, Interval, Ratio
- Discovering patterns: Trends, Associations, Clusters, Outliers
- Data visualization techniques: Bar Charts, Line Graphs, Pie Charts, Histograms
- Data preprocessing pipeline: Cleaning, Integration, Transformation, Reduction, Discretization

---

### Unit 2 — Market Basket Analysis & Association Rule Mining
**File:** `Unit2.pptx`

Explores transactional data patterns and the algorithms used to mine them.

- What is Market Basket Analysis and its business applications
- Key metrics: Support, Confidence, and Lift
- The **Apriori Algorithm**: candidate generation, pruning, and rule extraction
- **FP-Growth Algorithm**: compact FP-tree construction without candidate generation
- Vertical data format and TID-list intersection approach
- Closed Frequent Itemsets to reduce pattern redundancy
- Multilevel Association Rules using concept hierarchies
- Multidimensional and correlation-based rule analysis

---

### Unit 3 — Decision Trees in Data Mining
**File:** `Unit3.pdf`

A deep dive into decision tree-based classification and prediction.

- What are Decision Trees: supervised learning, tree structure, interpretability
- Anatomy: Root Node, Internal Nodes, Branches, Leaf Nodes
- Decision Tree Induction via recursive partitioning (greedy, top-down)
- Attribute Selection Measures: Information Gain, Gain Ratio, Gini Index
- **ID3 Algorithm**: uses Information Gain; pros & cons
- **CART Algorithm**: binary splits, Gini-based, supports regression
- Tree Pruning: Pre-Pruning vs. Post-Pruning
- Real-world applications: Medical Diagnosis, Fraud Detection, Customer Segmentation

---

### Unit 4 — Hierarchical Clustering in Data Mining
**File:** `Unit4.pdf`

Covers cluster analysis through hierarchical methods and dendrogram interpretation.

- What is Hierarchical Clustering and how it differs from partitioning methods
- **Agglomerative (Bottom-Up)**: starts with individual points, merges step by step
- **Divisive (Top-Down)**: starts with one cluster, recursively splits
- Linkage Methods: Single Linkage, Complete Linkage, Average Linkage
- Reading and interpreting Dendrograms
- Advantages: No predefined K, visual interpretability, pattern discovery, domain versatility
- Limitations: O(n³) complexity, greedy & irreversible decisions, noise sensitivity

---

### Unit 5 — Outlier Detection and Data Mining Applications
**File:** `Unit5.pdf`

Examines techniques for identifying anomalies and applying data mining in real-world domains.

- What is an Outlier: measurement errors, data entry mistakes, genuine rare events
- Challenges: large datasets, high dimensionality, noise, dynamic data streams
- Detection approaches: **Supervised**, **Semi-Supervised**, **Unsupervised**
- Statistical methods: Mean & Standard Deviation, Z-Score, Probability Distribution Models, Hypothesis Testing
- **Data Mining in Recommender Systems**: behaviour analysis, similarity matching, predictive personalization
- **Data Mining Applications**: Intrusion Detection, Financial Analysis & Fraud Detection
